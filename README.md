# Damascus International Fair Management System 🏛️

A robust and scalable backend system designed to manage the operations of the Damascus International Fair. This project serves as a graduation project, focusing on modern software architecture and containerized environments.

## 🚀 Features
- **Exhibitor Management:** Streamlined process for company registrations and booth assignments.
- **Media Handling:** Integrated S3-compatible cloud storage for uploading company assets and logos.
- **Dynamic Settings:** Global system configuration managed via seeders and cache for high performance.
- **API-First Architecture:** Fully documented RESTful APIs for mobile and web integration.
- **Containerized Environment:** Ready-to-go development environment using Docker Sail.

## 🛠️ Tech Stack
- **Framework:** Laravel 11 (PHP 8.4)
- **Database:** PostgreSQL
- **Environment:** Docker (Laravel Sail)
- **Caching:** Redis / Database Cache
- **Storage:** AWS S3 / Local Storage
